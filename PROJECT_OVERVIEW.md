# Project Overview

Project: Carousel Simulation
Version: v0.1.0

## Summary
Interactive Three.js simulation of crowd flow around a central pole with emergent behaviors, visual effects, and configurable physics.

## Core Features
- Particle crowd simulation with orbit/seek/leave states
- Group-based behaviors and attraction
- Trails, pheromones, bloom, and optional depth of field
- Interactive obstacle dragging
- GUI controls for visual and physics tuning

- Three.js
- Vite + TypeScript
